How It Works

3단계 디자인 지능 파이프라인

크롤링부터 UI 생성까지. 학습된 지식은 영구적으로 축적됩니다.

STEP 01
🕸️

Crawl & Learn

Playwright로 Awwwards급 사이트를 크롤링합니다. CSS, DOM, 스크린샷을 수집하고 캐시합니다.

npx auradkit-trainer learn --all
→
STEP 02
🧠

7-Axis Analysis

Visual Hierarchy, Spatial Rhythm, Typography 등 7개 디자인 축으로 0–10점 정밀 채점합니다.

npx auradkit-trainer status
→
STEP 03
✨

Compile & Generate

학습 데이터를 SKILL.md로 압축합니다. Claude Code가 이 지식으로 역대급 UI를 생성합니다.

npx auradkit-trainer compile
7 Design Axes

모든 디자인 결정은 데이터 기반

최고 수준 사이트에서 추출한 7개 측정 축. 각 축마다 10점 만점으로 정밀 채점합니다.

CLI Demo

실제로 어떻게 작동하나요?

3개의 명령으로 50개 사이트의 디자인 지식을 학습하고 UI를 만듭니다.

auradkit-trainer — zsh
Get Started

30초면 시작할 수 있습니다

Node.js 20+와 Claude Code만 있으면 됩니다.

npm install -g auradkit-trainer
